# SimpleMidiController

Welcome to SimpleMidiController support page.

SimpleMidiController is a MIDI controller app for iPhone and iPad. Control synthesizers and DAWs via Bluetooth MIDI, USB MIDI, or Virtual MIDI port.

---

## Features

### Bluetooth MIDI
Connect wirelessly to Bluetooth MIDI devices. No cables, no adapters â€“ just pair and play.

### USB MIDI
Connect to USB MIDI devices using the Lightning to USB adapter or USB-C.

### Virtual MIDI
Send MIDI to other apps on the same device using Virtual MIDI ports.

### Simple Interface
Clean, intuitive controls designed for live performance and studio use.

---

## Quick Start

1. **Launch the app**
2. **Connect your MIDI device** (Bluetooth, USB, or Virtual MIDI)
3. **Start controlling** your synthesizers and DAWs

# Buttons, Sliders, and Pads — Three Control Types

SimpleMidiController offers three versatile control types — simple yet powerful enough for any MIDI workflow.

## Buttons

Send Note On/Off or Control Change (CC) messages.

- Instant MIDI message on tap
- Perfect for program changes and scene switching
- Great for live triggering

## Sliders

Send continuous Control Change values.

- Ideal for volume, pan, filter, and other continuous parameters
- Smooth value changes that follow your finger
- Works as an expression controller

## Pads

Velocity-sensitive triggers.

- Velocity changes based on tap intensity
- Use as drum pads or sampler triggers
- Intuitive, performance-ready

## From Stage to Studio

These three control types cover everything from live performance to studio DAW control. The simple interface keeps you focused on making music, not fighting your tools.

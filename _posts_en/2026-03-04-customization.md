# Customize to Fit Your Style

Every control in SimpleMidiController can be tailored to match your performance setup.

## Per-Control Settings

Each button, slider, and pad offers these configuration options:

### MIDI Channel (1-16)

When using multi-timbral instruments or multiple synths, assign different channels to different controls. Operate multiple sound sources from a single screen.

### Control Number / Note Number

Specify which MIDI message to send. Set note numbers or CC numbers for buttons, CC numbers for sliders.

### Value Range

Customize the minimum and maximum values for sliders. Useful when you want to limit a filter cutoff to a specific range, for example.

## Global Settings

### Default MIDI Channel

Set a default channel applied to all controls. Individual settings override this when specified.

### Velocity Curve

Adjust pad response sensitivity. Make soft touches produce high velocity, or widen the dynamic range for more expressive playing.

## Make It Yours

Customize SimpleMidiController for your gear and songs, and it becomes your personal MIDI controller.
